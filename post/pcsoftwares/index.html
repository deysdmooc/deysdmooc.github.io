计算机常用软件推荐

2021-01-23 3 min read

工欲善其事，必先利其器。针对工作中常见的场景，我们精心挑选了一些优质软件分享给大家。
本文内未标明下载地址的软件均可在腾讯电脑管家内搜索下载。

系统类

杀毒软件

腾讯电脑管家（点击下载）

杀毒功能够用，工具箱充实，最难能可贵的是不但自身广告极少，还可以消灭其他应用的弹窗广告。

压缩软件

BandiZip

界面美观，使用便捷，美中不足的是解压效率较低。

7Zip

1.4M的超小身材却有着极快的解压速率，且完全无广告。但操作略显繁琐，适合经常处理大型文件的人。

浏览器

Microsoft Edge

Windows 10自带浏览器，使用Chromium内核速度极快，无广告，可用微软账号同步收藏夹，插件等。缺点是没有IE内核，个别老旧网站无法正常访问。建议搭配IE浏览器使用。

Google Chrome

适用于非Windows 10的系统，速度极快，无广告。缺点同Edge浏览器。

本地视频播放

PotPlayer

Windows平台最强大的本地视频播放软件。支持格式多，解码能力强大，支持12倍变速，无广告。

桌面整理

腾讯桌面整理

腾讯电脑管家工具箱内点击即可使用。占用低，界面美观。支持一键整理。

去广告

软件弹窗拦截

腾讯电脑管家工具箱内点击即可使用。可去除绝大多数弹窗广告。针对少数顽固弹窗可以手动截图添加。

效率类

清单

Microsoft TO DO

需在Windows 10自带应用商店内搜索下载。微软出品的免费清单软件，支持PC，iOS，Android平台。界面清爽，基本功能齐全，推送即时。配合GTD时间管理方法使用更佳。

云笔记

有道云笔记

免费，稳定，空间大的笔记软件别无分店。适合备份重要的资料。

本地搜索

Everything

快速全盘搜索本地文件，再也不用担心找不到藏在某个文件夹角落的文件了。

云盘

Microsoft OneDrive

免费用户可使用5GB容量。上传下载速度不俗。可当同步盘使用。
